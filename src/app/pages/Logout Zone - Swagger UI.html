Superadmin

Allows us to add new superadmin to the system.

Request Body

  • fullName: Full name of the superadmin. Required
  • email: Email of the superadmin. Required
  • password: Password of the superadmin. Required
  • mobile: Mobile number of the superadmin. Required
  • role: Role of the superadmin. Required
  • isDeleted: Flag indicating whether the superadmin is deleted or not. Optional. Default is False.

Response

  • return: Returns a message indicating whether the superadmin signup has been created or not.
